# Weather Dashboard

A modern weather dashboard built using React, Vite, Tailwind CSS, Recharts, and Open-Meteo APIs.

## Features

- City Weather Search
- Current Location Weather
- Dark / Light Mode
- Recent Searches
- Favorite Cities
- Weather Conditions
- 5-Day Forecast
- Temperature Trend Charts
- Compare Cities
- Loading Skeletons

## Tech Stack

- React
- Vite
- Tailwind CSS
- Recharts
- Open-Meteo API
- Local Storage
